# Sber Smart Home для Home Assistant

Интеграция для управления умными устройствами Сбер через Home Assistant.

## Возможности

- Поддержка ламп и умных лент (включение, яркость, цвет, температура)
- Поддержка реле и умных розеток
- Поддержка датчиков (температура, влажность, мощность, напряжение, ток)
- Автоматическое обновление состояния устройств
- Интеграция с автоматизациями Home Assistant

## Установка через HACS

### Вариант 1: Из HACS Repository (рекомендуемый)

1. Откройте HACS в Home Assistant
2. Перейдите в "Репозитории"
3. Нажмите "+" и добавьте URL репозитория:
   ```
https://github.com/rekillicloud/ha_sber_smart_home
   ```
4. Установите интеграцию "Sber Smart Home"

### Вариант 2: Ручная установка

1. Скопируйте папку `custom_components/sber_smart_home` в директорию `config/custom_components/` вашего Home Assistant
2. Перезапустите Home Assistant

## Настройка

1. Перейдите в "Устройства и службы" → "Добавить интеграцию"
2. Найдите "Sber Smart Home"
3. Следуйте инструкции по авторизации:
   - Откроется ссылка для авторизации в Сбер ID
   - Войдите в аккаунт
   - Скопируйте `code` из URL после редиректа
   - Вставьте код в поле интеграции

## Требования

- Home Assistant 2024.1 или новее
- Python 3.11+
- Аккаунт Сбер с подключенными умными устройствами

## Поддерживаемые устройства

- Умные лампы Sber
- Умные ленты Sber
- Умные розетки Sber
- Датчики температуры и влажности
- Контроллеры умного дома
